# Serverless Web Application on AWS

## Project Overview
This project is a Serverless Web Application built using Amazon Web Services (AWS). The application allows users to manage employee records using AWS serverless services.

## Technologies Used
- HTML
- CSS
- JavaScript
- AWS Lambda
- Amazon API Gateway
- Amazon DynamoDB
- Amazon S3
- Amazon CloudFront

## Features
- Add Employee
- View Employee Details
- Delete Employee
- Serverless Architecture
- Responsive Web Interface

## AWS Services Used
- Amazon S3
- AWS Lambda
- Amazon API Gateway
- Amazon DynamoDB
- Amazon CloudFront

## Project Status
Completed as a Final Year B.Tech Project.
